## Simple timer

Built on pure HTML + CSS + JS with web workers (in order to work when tab is idle)

### Development

Since you cannot run a web worker locally you should run this site on your localhost - the easies way to do it is to use `http-server` - https://stackoverflow.com/questions/38497334/how-to-run-html-file-on-localhost#:~:text=You%20can%20run%20your%20file%20in%20http%2Dserver.
